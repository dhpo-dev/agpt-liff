AGPT サテライト協賛 利用規約

第1条（目的）

本規約は、AGPT（以下「主催者」）が運営するポーカートーナメントサテライト事業における協賛契約の条件を定めるものです。

第2条（協賛内容）

協賛店舗（以下「店舗」）は、選択したプランに応じた協賛金を支払い、主催者より招待チケットおよび各種特典の提供を受けます。

第3条（招待チケットの取り扱い）

店舗は、主催者から付与された招待チケットについて、以下の事項を遵守するものとします。

  1. 招待チケットは非売品であり、売買・再販売を禁止します
  2. 第三者への譲渡を禁止します
  3. 換金を禁止します
  4. チケットはAGPT主催トーナメントでのみ使用可能です
  5. チケットの有効期限は発行日から6ヶ月間です
  6. 紛失・盗難時は一律無効とし、再発行は行いません

第4条（チケット配布・付与報告）

店舗は、チケットの配布方法を自由に決定できます。ただし、チケットを配布した場合は、以下の情報を速やかに主催者に報告する義務を負います。

  1. チケットID
  2. 付与先プレイヤーのWaitinglist ID
  3. 付与日

第5条（禁止事項）

店舗は以下の行為を行ってはなりません。

  1. チケットの有償販売・オークション出品
  2. 架空の人物へのチケット付与（名義貸し）
  3. 主催者の承認なく第三者にチケットを譲渡する行為
  4. 虚偽の付与報告
  5. 主催者のブランドイメージを毀損する行為

第6条（協賛金・支払い）

協賛金は、契約成立後に主催者が指定する方法により支払うものとします。入金確認後にチケットの発行・配送手続きを行います。

第7条（契約期間）

本契約はシーズン単位で締結し、契約期間中の途中解約による返金は原則として行いません。

第8条（契約解除）

主催者は、店舗が本規約に違反した場合、事前の通知なく契約を解除し、未使用チケットを無効化できるものとします。この場合、協賛金の返還は行いません。

第9条（免責事項）

天災・感染症・行政指導等の不可抗力により大会が中止となった場合、主催者は代替大会の開催または協賛金の一部返還について誠意をもって協議するものとします。

第10条（個人情報の取り扱い）

主催者は、本申込で取得した個人情報を、協賛契約の履行、チケット管理、大会運営の目的にのみ使用し、法令に基づく場合を除き第三者に提供しません。

第11条（Waitinglistシステムの利用）

店舗は、協賛期間中Waitinglistシステムの利用契約を維持するものとします。Waitinglistシステムはチケット管理・付与報告・参加者管理に使用されます。

第12条（準拠法・管轄）

本規約は日本法に準拠し、紛争が生じた場合は東京地方裁判所を第一審の専属的合意管轄裁判所とします。
